Detailed Description

This is the library for the Adafruit Motor Shield V2 for Arduino. It supports DC motors & Stepper motors with microstepping as well as stacking-support. It is not compatible with the V1 library.

◆ MicroSteps

enum Adafruit::MicroSteps : uint16_t

Defines the number of microsteps executed per step of a stepper motor. Increasing microsteps per step limits the maximum RPM achievable by a stepper motor due to I2C bus constraints. Upper limits for each microstep for a 200 steps/revolution, double coil stepper motor are provided.

Enumerator
STEP8 

8 microsteps per step, max speed 10 RPM.

STEP16 

16 microsteps per step, max speed 5 RPM.

STEP32 

32 microsteps per step, max speed 2.5 RPM.

STEP64 

64 microsteps per step, max speed 1.25 RPM.

STEP128 

128 microsteps per step, max speed 0.625 RPM.

STEP256 

256 microsteps per step, max speed 0.3125 RPM.

STEP512 

512 microsteps per step, max speed 0.15625 RPM.
